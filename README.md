# LLM-Threat-Intel-Summariser

# 🧠 LLM Threat Intel Summariser

An automated threat intelligence pipeline that ingests feeds from OTX AlienVault and cybersecurity RSS sources, then uses the Claude API (LLM) to generate concise, actionable daily threat briefings tailored for BFSI security teams.

---

## 📂 Repository Structure

```
llm-threat-intel-summariser/
├── README.md
├── requirements.txt
├── config.py                       # API keys and configuration
├── src/
│   ├── fetch_otx.py                # OTX AlienVault feed ingestion
│   ├── fetch_rss.py                # RSS cybersecurity feed ingestion
│   ├── summarise.py                # Claude API summarisation engine
│   ├── report_generator.py         # Markdown report generator
│   └── main.py                     # Orchestrator — run this daily
└── reports/
    └── sample-report-2026-04-01.md # Example generated report
```

---

## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/its-me-anvesh-var/llm-threat-intel-summariser
cd llm-threat-intel-summariser
pip install -r requirements.txt

# Set API keys
export OTX_API_KEY="your_otx_key_here"
export ANTHROPIC_API_KEY="your_claude_key_here"

# Run the pipeline
python src/main.py

# Report saved to reports/threat-brief-YYYY-MM-DD.md
```

## 🔄 Pipeline Architecture

```
OTX AlienVault API          RSS Feeds
(latest pulses)         (CERT-In, Krebs, etc.)
        │                       │
        └───────────┬───────────┘
                    ▼
           Raw Threat Data
           (IOCs, CVEs, TTPs)
                    │
                    ▼
           Claude API (LLM)
           (summarise + prioritise)
                    │
                    ▼
         Daily Threat Brief (Markdown)
         + IOC List (CSV)
         + Priority Alerts
```
